## 1. 成長與經驗

| 項目 | 公式 | 說明 |
|---|---|---|
| 升級所需經驗 | `⌊(30 × 等級^2 + 40) × 轉生倍率⌋` | 升到「下一級」所需的經驗值；等級最高 9,999。轉生倍率：0轉 1 倍；每次轉生為上一次的 10 倍，10轉為 `10^10` 倍。 |
| 等級基礎四維 | `5 + (等級-1) × 2` | 力量 / 敏捷 / 智力 / 耐力 四項相同，每升 1 級各 +2 |
| 技能點總預算 | 初始 `2`；未轉生時每升 1 級 `+1`，最高 `10,000` | 初始兩個 1 級技能也計入預算；已使用 = 所有技能等級總和；可用技能點 = 總預算 - 已使用。 |
| 技能裝載欄 | `clamp(⌊等級 / 20⌋, 2, 20)` | 每 20 級 +1 格，最少 2 格、最多 20 格 |

### 1.1 轉生

- 角色達到 9,999 級後可轉生，最高 10 轉；轉生後等級與當前經驗重置為 1 級／0 經驗。
- 轉生保留裝備、技能、資源與關卡進度。
- 階級依序為冒險者、勇者、大劍師、破世者、不朽者、王者、大主宰、神聖尊者、大聖王、至高主宰、位面創世神。
- 生命、法力與四維在原始總值完成後套用最終倍率：1～10 轉為 ×10、×20、×40、×80、×160、×320、×640、×1280、×2560、×5120。
- 轉生後升級經驗需求倍率為上次轉生的10倍。
- 玩家升級經驗需求：`xpForLevel(l) = floor((30 × l^2 + 40) × 轉生經驗倍率)`，其中 l 為目前等級。
- 轉生後每升一級改獲得 1 點轉生天賦點；技能點不再增加，天賦點目前只保存待後續系統使用。
- 一般技能上限為 `20 + 轉生次數 × 10`（被動技能基礎 30）；融合技能在其原始上限上再加 `轉生次數 × 20`。
- 神鑄系統在 **0 轉達到 1,000 級**時開放；開放後永久保留，轉生不會關閉。

## 2. 玩家屬性（50+ 屬性派生）

### 2.1 彙總流程

```
等級基礎四維
  → 累加：裝備詞條（× 強化倍率）＋ 被動技能（每級效果 × 技能等級）
        ＋ 鑲嵌寶石（× (1 + 寶石鑲嵌效率%)）＋ 附魔（元素附傷 / 抗性 / 功能）
  → 派生出面板上所有數值（多數有上限）
```

- 詞條「攻擊速度%」計入攻速加成桶；「火焰抗性%」等六系計入元素抗性桶。
- **強化倍率**：裝備每 +1，其所有詞條數值 ×(1 + 0.05×強化等級)。
- 融合寶石（雙屬性）逐屬性計入，同樣吃寶石鑲嵌效率放大。

### 2.2 基礎屬性派生公式

| 屬性 | 公式 |
|---|---|
| 生命上限 | `(120 + (等級-1)×22 + 耐力×10 + 生命定值) × (1 + 生命%)` |
| 生命恢復/秒 | 詞條/寶石加成值（另有野外基礎回復，見 §3.5） |
| 法力上限 | `40 + 智力×4 + 法力定值` |
| 法力恢復/秒 | `2 + 智力×0.06 + 加成` |
| 物理攻擊 | `(8 + (等級-1)×1.6 + 力量×2 + 物攻定值) × (1 + 物攻%)` |
| 魔法攻擊 | `(6 + (等級-1)×1.2 + 智力×2 + 魔攻定值) × (1 + 魔攻%)` |
| 物理防禦 | `(4 + (等級-1)×1.0 + 耐力×0.9 + 物防定值) × (1 + 物防%)` |
| 魔法防禦 | `(3 + (等級-1)×0.8 + 智力×0.7 + 魔防定值) × (1 + 物防%［共用物防%詞條］)` |

## 3. 戰鬥核心

### 3.1 防禦減傷（核心公式）

```
減傷率 = 防禦 / (防禦 + 60 + 8 × 攻擊者等級)
```

防禦收益遞減；攻擊者等級越高，同樣防禦的減傷率越低。當有效防禦 `≤ 0` 時，減傷率直接視為 `0%`，等同於不再享有防禦減傷。

技能或減益造成的破防先計算有效防禦：

`有效防禦 = 防禦 × (1 - 技能防禦降低% / 100) × (1 - 破甲%) × (1 - 穿透%)`

實際程式中的技能防禦降低會以 `防禦 × (1 - 防禦降低% / 100)` 計算；當降低超過 100% 時會得到負防禦，但後續仍依 `有效防禦 ≤ 0` 處理為 0% 減傷，不會產生額外負防禦傷害。

### 3.2 單次攻擊結算順序（resolveHit）

```
1. 命中判定：命中機率 = clamp(攻擊者命中% - 防守者閃避%, 5%, 100%)，未命中 → MISS
2. 基礎傷害：
   物理部分 = 物攻 × (1 - 防禦減傷(有效物防)) × (1 - 物理抗性%)
   魔法部分 = 魔攻 × (1 - 防禦減傷(有效魔防)) × (1 - 魔法抗性%)
   其中 有效防禦 = 防禦 × (1 - 破甲%) × (1 - 穿透%)
   ※ 玩家普攻為「雙修型」：物理+魔法兩部分同時結算相加
3. 傷害浮動：× rnd(0.9, 1.1)
4. 暴擊判定：機率 = 暴擊率，成功 → × 暴擊傷害%（基礎 150%）
5. 元素附加：每系傷害 = 元素附傷 × (1 - 對應元素抗性%)，不重複套用魔法抗性，並觸發元素特效（§3.3）
6. 真實傷害：+ 攻擊力 × 真傷%（無視防禦、抗性與格擋）
7. 對菁英 ×(1+對菁英傷害%)、對 BOSS ×(1+對BOSS傷害%)
8. 格擋判定：機率 = 格擋率（上限 50%），成功 → 傷害 × (1 - (30% + 格擋減傷%))，總減傷上限 85%
9. 神鑄特效「聖佑」：受到的所有傷害再按比例減免，上限 50%
10. 全局減傷：減傷率 = `min(85%, 全局減傷總合 / (全局減傷總合 + 20000))`（上限 85%），套用 `傷害 × (1 − 減傷率)`，在最低傷害與護盾結算前套用
11. 最低傷害 1 → 護盾優先吸收 → 扣血
12. 反震：攻擊方受到 防守方最大生命 × 反震% 的傷害（最低 1）
```

補充：一般物理/魔法傷害、元素附加傷害與持續傷害會依各自流程套用全局減傷；技能類型為「真實」的技能目前走直接扣血路徑，傷害為 `技能原始傷害 × rnd(0.95, 1.05)`，不經防禦、抗性、格擋與 `resolveHit` 的全局減傷流程。被動「真傷」則是在 `resolveHit` 中附加，會跟隨該流程。

### 3.3 元素特效（每次命中對每系元素獨立判定）

| 元素 | 觸發機率 | 效果 |
|---|---|---|
| ❄️ 冰霜 | 15% | 敵人減速 2 秒（受控制時間縮減影響；可被控制抵抗擋下） |
| ⚡ 雷電 | 10% | 追加該次元素傷害 80% 的連鎖電擊 |
| ☠️ 劇毒 | 25% | 中毒：每秒受元素傷害 50% 的跳傷，持續 4 秒（無視防禦） |
| ✨ 聖光 | 20% | 淨化攻擊者自身全部負面狀態 |
| 🌑 暗影 | 100% | 汲取元素傷害的 25% 回復生命 |

### 3.4 狀態效果

- **減速**：攻速 ×0.7（-30%），即攻擊冷卻累積速度打七折。
- **暈眩**：期間完全無法行動（普攻與技能皆停）。
- **控制抵抗**：`ctrlRes%` 機率完全免疫本次暈眩/減速。
- **受控時間** = 原始時間 × (1 - 韌性%) × (1 - 控制時間縮減%)。
- **持續傷害（DoT）**：同名效果疊加取高（傷害與時間各取較大者）；中毒為獨立通道，其餘（流血/燃燒/詛咒/侵蝕）共用列表，全部無視防禦。

### 3.5 回復與護盾

| 項目 | 公式 |
|---|---|
| 野外基礎回復 | 每秒回復 `最大生命 × 1.5% + 生命恢復屬性`（高塔內只有生命恢復屬性與再生增益，無 1.5% 基礎） |
| 擊殺回復 | 野外每擊殺一隻回復 `最大生命 × 12%`（吸魂被動再另加 `最大生命 × 吸魂%`） |
| 溢出轉護盾 | 治療溢出部分的 **50%** 轉為護盾 |
| 護盾上限（治療轉化） | `最大生命 × 15% × (1 + 護盾效率%)` |
| 護盾上限（技能直接給予） | `最大生命 × 50%`；技能護盾量本身 ×(1 + 護盾效率%) |
| 吸血 / 吸魔 | 每次造成傷害回復 `傷害 × 吸血%` 生命 / `傷害 × 吸魔%` 法力 |

### 3.6 怪物攻擊玩家

- 怪物固定：暴擊率 5%、暴擊傷害 150%、命中 100%。
- `magic: true` 的怪物走**魔法傷害**（對玩家魔防），其餘走物理。
- 野外怪物初次出手延遲：`1/攻速 + 0.4` 秒。
- 普通關卡若同時出現多隻敵人，敵人各自獨立攻擊；玩家普攻固定鎖定第一隻存活敵人，該敵人死亡後才切換下一隻。
- 玩家死亡：退回第 1 階段，3 秒後復活（滿血滿藍、狀態清空）。

## 4. 敵方屬性

### 4.1 野外怪物成長曲線（依階段 s）

| 數值 | 公式 |
|---|---|
| 生命 | `(30 + s×8) × 1.13^(s-1)` |
| 攻擊 | `(6 + s×1.2) × 1.10^(s-1)` |
| 物理防禦 | `(2 + s×0.5) × 1.08^(s-1)` |
| 魔法防禦 | `物理防禦 × 0.75` |
| 物理/魔法減傷 | `防禦減傷率 = 防禦 ÷ (防禦 + 60 + 8 × 玩家等級)`；物理減傷使用物防，魔法減傷使用魔防；普通 NPC 沒有額外全局減傷 |
| 攻擊速度 | 0.75 次/秒 |
| 金幣 | `(5 + s) × 1.07^(s-1)` |
| 經驗 | `(8 + s) × 1.08^(s-1)` |

普通與菁英關卡每波敵人數量皆獨立擲骰：1 隻 60%、2 隻 25%、3 隻 10%、4 隻 5%。

### 4.2 菁英與戰鬥場景

- **菁英**：階段為 **10 的倍數** 時出現，固定單一敵人。生命 ×2.5、攻擊 ×1.5、金幣/經驗 ×3、閃避 5%、攻速 1.25。
- 同一波中的每隻普通敵人各自獨立計算經驗、金幣、裝備、寶石、附魔書、精華、魔塵與零件掉落；整波敵人全部死亡後才推進下一階段。
- **場景倍率**（乘在上表結果之上；裝備掉落表不受影響）：

| 場景 | 生命 | 攻擊 | 防禦 | 經驗/金幣/材料 | 解鎖條件 |
|---|---|---|---|---|---|
| 🌿 草原 | ×1 | ×1 | ×1 | ×1 | — |
| 🏜️ 荒漠 | ×2.2 | ×1.8 | ×1.6 | ×2 | 草原最高 100 階 |
| 🦠 沼澤 | ×4 | ×2.8 | ×2.4 | ×3 | 荒漠最高 100 階 |

### 4.3 高塔 BOSS（依樓層 f）

| 數值 | 公式 |
|---|---|
| 對應野外階段 | `4 + f×5`（以此階段的野外怪為基準） |
| 等級 | `對應階段 + 3` |
| 挑戰金幣消耗 | `round(100,000 × f^2.6)`；開始挑戰前扣除，失敗或撤退不退回 |
| 生命 | `基準生命 × 22`；地獄之塔再 ×5 |
| 攻擊 | `基準攻擊 × 1.9`；地獄之塔再 ×2 |
| 物/魔防 | `基準物/魔防 × 1.5` |
| 攻速 / 閃避 / 控制抵抗 | 2.0 次/秒 / `min(5 + f, 20)%` / 70% |
| 元素附傷（元素 BOSS） | `基準攻擊 × 0.5`（以魔法攻擊結算） |
| 物理/魔法減傷 | `防禦減傷率 = 防禦 ÷ (防禦 + 60 + 8 × 玩家等級)`；BOSS 沒有額外全局減傷 |
| 經驗 | `該樓層建議野外階段的普通怪物經驗 × 2`；再套用玩家經驗加成 |
| 魔魂本源 | 僅地獄之塔：第 51 層 5%，之後每高 1 層 +1%；獨立判定 1 顆 |

**戰鬥規則**：限時 **60 秒**；**40 秒**時若 BOSS 血量 > `50% + 玩家狂暴閾值屬性` → 狂暴（傷害 ×3，即 +200%）；每 **8 秒**蓄力重擊（傷害 ×2.2，狂暴時 ×2.2×3）。

## 5. 掉落與獎勵

### 5.1 掉落表通則

- 掉寶率平衡：裝備詞條、附魔、神鑄「貪婪」、技能「尋寶直覺」與相關分解零件的掉寶率效果，實際計算一律乘以 `0.5`。存檔保留原始數值，既有物品也直接套用此折半規則。
- 目前分解槽中會折半的掉寶相關零件為：精粹透鏡、太古精華萃取器、複製處理艙、幸運晶片、寶石篩選器、拓本回收臂、探礦核心；碎片／金幣產量、速度與寶石提純等非掉寶率效果不受影響。

- 每個品質**獨立擲骰**，可同時掉多件。
- 機率 > 100% 的「必掉＋餘數」規則：`件數 = ⌊機率/100⌋ + (機率餘數% 的機率再 1 件)`。
  例：150% = 必掉 1 件 + 50% 機率第 2 件。
- 實際機率 = 表值 × (1 + 有效掉寶率%)；野外菁英再 ×1.5。
- 掉落表本體（`FIELD_DROP_TABLE` 野外 4 檔 / `BOSS_DROP_TABLE` 高塔 7 檔）在 `data.js`。

野外裝備掉落表（索引 0～7 = 普通、精良、稀有、獨特、史詩、傳說、神話、創世）：

| 怪物等級 | 各品質基礎機率（%） |
|---|---|
| 1～49 | [25, 10, 5, 0, 0, 0, 0, 0] |
| 50～99 | [25, 10, 5, 2, 0, 0, 0, 0] |
| 100～149 | [0, 20, 15, 10, 5, 2, 0, 0] |
| 150+ | [0, 0, 25, 15, 5, 2, 0.25, 0] |

野外裝備實際機率 = 表值 × `(1 + 有效掉寶率% / 100)`；菁英敵人再乘 1.5。荒漠與沼澤的場景倍率不作用於裝備掉落，只作用於非裝備材料與經驗／金幣。

### 5.2 野外材料掉落

| 材料 | 條件 | 機率 / 公式 | 數量/等級 |
|---|---|---|---|
| 💎 寶石 | 階段 4+ | `6% × (1+掉寶率%) × 場景倍率` | 等級：基準級 = `clamp(1 + ⌊階段/15⌋, 1, 5)`，70% 出基準級、30% 低一級 |

## 6. 裝備數值

### 6.1 生成規則

- 部位：隨機 10 種之一；等級 = `階段 ± 1`（最低 1）。
- 詞條數：依稀有度表（普通 1~2 → 創世 7）；玩家有「詞條上限率」時，`詞條上限率%` 機率 +1 條（總上限 8 條）。
- 詞條種類：依詞條池權重抽取（高階詞條有最低稀有度 `minR`、部位限定 `slots` 門檻），不重複。
- 幸運值：每條詞條 `幸運/2%` 機率重骰一次數值取較佳值。
- 稀有級（含）以上附帶 1 個特殊被動。

### 6.3 太古詞條

- 只有等級 `>=200`、史詩級以上裝備會在生成時判定太古詞條；普通、精良、稀有裝備不會出現太古詞條。自然掉落使用來源敵人的規則；神鑄成功產物則使用一般敵人的規則，並以素材最高等級作為敵人等級。合成與 GM 直接生成仍不套用自然掉落判定。
- 野外每個詞條獨立判定：`clamp(2% + (敵人等級-250)×0.2%, 2%, 5%)`。
- 高塔 BOSS 每個詞條獨立判定：`BOSS等級>=40` 時為 `min(100%, 20% + (BOSS等級-40)×2%)`。
- 洗煉時也只有史詩級以上裝備可產生太古詞條；太古裝備再次洗煉仍可能回到普通詞條。
- 太古詞條數值固定為原洗煉上限（基準值 × 1.2）再乘 `1.35`；也就是原詞條上限的 135% 強度。
- 顯示時詞條名稱、數值與前方標記皆使用橘色，標記為較大的六角星 `✡`。

### 6.4 詞條數值

```
基準值 = (base + base × lv係數 × (裝備等級 - 1)) × 稀有度倍率
實際值 = 基準值 × rnd(0.8, 1.2)      ← 即洗煉區間為基準值的 80% ~ 120%
```

- `base`、`lv係數` 見 `data.js` 的 `AFFIX_POOL`；稀有度倍率：普通 1.0 / 精良 1.35 / 稀有 1.75 / 獨特 2.3 / 史詩 3.0 / 傳說 4.0 / 神話 5.2 / 創世 6.8 / 神鑄創世 10.2。
- **全局減傷詞綴**：定值型 `base=3、lv係數=0.35`，最低史詩稀有度，無部位限制，因此所有裝備部位均可出現；多件裝備直接加總。

### 6.5 特殊被動數值

```
被動數值 = base + perR × (稀有度 - 2)      ← 稀有=基準，每高一階 +perR
```

| 被動 | base / perR | 上限（多件疊加後） |
|---|---|---|
| 破甲 | 10 / 2 | — |
| 反震 | 5 / 1 | — |
| 連擊 | 10 / 2 | 45% |
| 暈眩 | 6 / 1.5 | 30% |
| 減速 | 12 / 3 | — |
| 真傷 | 6 / 1.5 | — |
| 吸魂 | 5 / 1.5 | — |

神鑄創世特效「神力」不直接加總到物理／魔法攻擊百分比，而是作為額外乘區：
`攻擊基礎 × (1 + 原有攻擊% / 100) × (1 + 神力% / 100)`；多件神力效果彼此相乘。

### 6.6 附魔數值

```
攻擊類（元素附傷）= (5 + 裝備等級×1.2) × 稀有度倍率 × (1 + 0.15×寶石等級)
                     火焰附魔再 ×1.25（定位：純高額傷害）
防禦/功能類（%）  = min(8 + 稀有度×4 + 寶石等級×3, 60)
```

- 附魔欄位數依稀有度：普~稀有 1、獨特~傳說 2、神話/創世 3。
- 手動附魔費用：1 本書 + 5 附魔精華；取下返還書、不退精華。
- 「寶石等級」參數只在生產線混合合成時帶入（= 消耗寶石等級 × (1+寶石鑲嵌效率%)），手動附魔為 0。

### 6.7 裝備戰力評分（自動換裝依據）

```
評分 = Σ(詞條值 × 權重 × 強化倍率)
     + Σ(鑲嵌寶石值 × 對應詞條權重)
     → 有特殊被動 ×1.15
     + Σ(附魔值 × 1.2［攻擊類］ or × 2［防/功能類］)
     → × (1 + 稀有度 × 0.06)
```

權重表 `SCORE_WEIGHTS` 見 `formula.js` §6（例：攻速 2.8、物攻% 2.6、暴擊率 2.4、生命定值僅 0.16）。
自動換裝條件：新裝評分 > 舊裝評分 × 1.02（避免頻繁換裝）。

### 6.8 分解產出

```
碎片 = max(1, (2 + 裝備等級×0.6) × 分解係數 × rnd(0.85, 1.15))
金幣 = (3 + 裝備等級) × 分解係數 × 0.5
精粹提取（機率見 §7.3）：精華 = 1~2 + ⌊稀有度/2⌋，另 30% 附贈 1 顆一級寶石
每個附魔額外回收 1 精華；鑲嵌的寶石（含融合寶石）分解前自動取回
```

分解係數（`RARITIES.salv`）：普通 1.0 → 精良 1.7 → 稀有 2.8 → 獨特 4.5 → 史詩 7.5 → 傳說 12 → 神話 19 → 創世 30。

拆解取得太古精華：史詩 0.5%、傳說 2%、神話 10%、創世 100%、神鑄創世 100%，每件裝備獨立判定 1 顆。

### 6.9 神鑄

| 項目 | 公式 / 規則 |
|---|---|
| 裝備神鑄材料 | 6 件同品質裝備；可用品質為傳說、神話、創世；神鑄創世為專屬最高品質 |
| 裝備神鑄成功率 | `clamp(該品質基礎成功率 + 魔塵數×5%, 0, 100)`；基礎值：傳說 55%、神話 40%、創世 25% |
| 裝備神鑄金幣 | 傳說 5,000,000；神話 20,000,000；創世 100,000,000 |
| 寶石神鑄 | 6 顆同種類同階寶石（5～9階）→ 1 顆高一階，最高 10 階 |
| 寶石神鑄成功率 | `clamp(該階級基礎成功率 + 魔塵數×3%, 0, 100)`；基礎值 5階 45%、6階 35%、7階 25%、8階 15%、9階 5% |

## 7. 強化 / 洗煉 / 分解 / 合成

### 7.1 強化

| 項目 | 公式 |
|---|---|
| 效果 | 每 +1：全詞條數值 ×(1 + 0.05×強化等級) |
| 金幣費用 | `25 × 1.45^目前強化等級 × (1 + 裝備等級×0.08)` |
| 碎片費用 | `8 × 1.35^目前強化等級 × (1 + 裝備等級×0.04)` |
| 基礎成功率 | +5 以內 100%；之後 `max(30%, 100% - (目標等級-5)×6%)` |
| 實際成功率 | `min(100%, 基礎 + 強化成功率屬性)` |
| 失敗懲罰 | 消耗半數費用，強化等級不變 |

### 7.2 洗煉（重骰詞條）

| 項目 | 公式 |
|---|---|
| 金幣費用 | `40 × 1.7^稀有度 × (1 + 裝備等級×0.15)` |
| 精華費用 | 普通～傳說：`1 + 稀有度`；神話：`9`；創世：`14`；神鑄創世：`20` |
| 整件洗煉 | 全部詞條重骰（種類+數值）；幸運值照 §6.1 取優 |
| 單詞條洗煉 | 同樣費用，指定詞條重骰為**其他種類**（不與現有詞條重複） |
| 使用太古精華 | 勾選後每次整件或單詞條洗煉消耗 1 顆太古精華；本次重骰的每個詞條獨立以 30% 機率成為太古詞條，原本的太古詞條也可能重骰回普通詞條 |

太古精華只影響本次洗煉的詞條結果，不改變金幣與附魔精華費用；資源不足時不會執行洗煉。

### 7.3 分解（生產線分解槽）

```
精粹提取率 = 10%（基礎） + 分解高產率屬性 + 幸運值/3 + 精粹透鏡零件（+3%/階）
分解速度：加速齒輪零件每階 +25%，現行規則再固定 +50 個百分點（T1~T7 = +75%~+225%；處理間隔 2 秒 ÷ (1+加成)）
```

**分解槽零件（13 種；分解槽使用金幣逐格解鎖，`data.js` `PART_TYPES` node:'salvage'；效果掛在 `factory.js` doSalvage）**

| 零件 | 效果（值 = perTier × 階級 T1~T7） |
|---|---|
| ⚙️ 加速齒輪 | 分解速度 +25%/階，另固定 +50 個百分點；T1~T7 = +75%~+225% |
| 🔥 碎片熔煉爐 | 分解碎片產量 +20%/階（乘算） |
| 💰 淘金濾網 | 分解金幣產量 +25%/階（乘算） |
| 🔬 精粹透鏡 | 精粹提取機率 +3%/階 |
| 🧪 精華凝結線圈 | 精粹提取時額外精華 +1/階 |
| 💎 寶石篩選器 | 每次分解 +0.6%/階 機率額外掉 1 級寶石 |
| 🔷 寶石提純器 | 精粹提取的寶石每級 +6%/階 機率再升 1 級（連續判定，上限 5 級；每次判定機率上限 90%，最高階寶石永遠是機率而非保證） |
| 📖 拓本回收臂 | 每次分解 +0.4%/階 機率回收 1 本附魔書 |
| ♻️ 複製處理艙 | 每次分解 +3%/階 機率碎片＋金幣翻倍 |
| 📚 知識回收器 | 每次分解 +1.5%/階 機率獲得經驗（裝備等級×25） |
| ⛏️ 探礦核心 | 每次分解 +0.15%/階 機率額外掉一個自動機組零件（階級隨裝備稀有度） |
| 🎰 幸運晶片 | 每次分解 +0.5%/階 機率「大豐收」：本次碎片/金幣/精華 ×3 |
| 🧬 太古精華萃取器 | 分解太古精華基礎機率乘以 `1 + 零件加成%`；T1~T7 分別為 +25%~+175% |

太古精華拆解判定：先以稀有度基礎機率乘上零件倍率判定；若裝備每有 1 條太古詞條，另獨立以 50% 判定 1 次，任一判定成功即取得 1 顆太古精華。基礎機率為史詩 0.5%、傳說 2%、神話 10%、創世與神鑄創世 100%。

分解槽解鎖：目前 10 格，最多 20 格。解鎖至第 N 格的費用序列為 `10,000 × 3^(N-1)`；例如目前 10 格時，下一格為第 11 格，費用為 `590,490,000`。介面最後追加擴充按鈕並顯示目前 `已解鎖/20`。舊存檔沒有解鎖欄位時，沿用既有 10 格。

倍率類（熔煉爐/濾網/複製/幸運晶片）互相乘算；`makePart` 改為「依節點均衡」挑選，避免 13 種分解零件稀釋合成零件（synth）的掉落率。

### 7.4 合成（生產線合成節點）

## 8. 寶石

### 8.1 能力數值（超線性成長）

```
數值 = base × 等級 × (1 + 0.2 × (等級-1))
```

即等級倍率為：Lv1 ×1、Lv2 ×2.4、Lv3 ×4.2、Lv4 ×6.4、Lv5 ×9。`base` 見 `data.js` 的 `GEM_TYPES`（**18 種**：12 攻擊/功能 + 6 防禦〔翡翠=韌性、綠松石=格擋率、瑪瑙=格擋減傷、珍珠=護盾效率、孔雀石=物理抗性、螢石=魔法抗性〕，2026-07-09 新增）。

### 8.2 插槽

- 插槽數依稀有度：普/精良 1、稀有/獨特 2、史詩 3、傳說 4、神話 5、創世 6。
- 鑲嵌效果吃「寶石鑲嵌效率%」全域放大。

### 8.3 寶石升階（工廠自動）與寶石合成（手動，2026-07-09 改版）

| 系統 | 公式 / 規則 |
|---|---|
| 寶石升階 | 3 顆同種同級 → 1 顆同種高一級（免費、自動；熔爐合成節點開關） |
| 寶石合成 | 2 顆「**同種類、同等級**」→ 1 顆**同種類下一階**；金幣費用依素材等級 100/300/900/2700/8100（無昇華機率，介面可選種類＋等級；選擇「全部類型寶石」時逐種類處理，不混合種類） |
| 合成換算 | 依 2:1 合成鏈，1 顆 N 級寶石 = `2^(N-1)` 顆 1 級（5 級 = 16 顆） |

### 8.3.1 寶石轉換（九宮格，2026-07-09 新增）

| 項目 | 規則 |
|---|---|
| 介面 | 3×3 = 9 格；每格放一種「同種類同等級」寶石，每格上限 **100 顆** |
| 轉換 | 一次性把格內所有寶石轉為選單所選種類；**同階轉換、數量不變**（例：紅1×10＋綠2×5＋紫3×3 → 黃1×10＋黃2×5＋黃3×3） |
| 限制 | **融合寶石無法轉換**；免費 |

### 8.3.2 寶石拆解（2026-07-09 新增）

| 項目 | 公式 |
|---|---|
| 一般寶石 | 1 顆 N 級（N≥2）→ `⌊2^(N-1) × 0.7⌋` 顆同種 1 級（**損失 30%**）：2級→1、3級→2、4級→5、5級→11 顆 |
| 融合寶石 | 1 階總成本 = `16 × 素材 5 階總數`（`leaves`，融合時雙方相加、視為 100% 成功）→ 產出 `⌊總成本 × 0.7⌋` 顆 1 級，依其屬性種類**均分**（餘數給第一種）。例：兩顆一般寶石融合（leaves=2）→ ⌊32×0.7⌋=22 → 各 11 顆；兩顆「融合 1 次」再融合（leaves=4）→ ⌊64×0.7⌋=44 |
| 1 階寶石 | 不可拆解 |

### 8.4 寶石融合 v2（僅限 5 階，產出雙屬性寶石）

| 項目 | 公式 / 規則 |
|---|---|
| 成功率 | `max(10%, 60% - 10% × 雙方融合次數之和)` |
| 融合次數 | **世代制**（2026-07-09 修正）：`max(雙方融合次數) + 1` — 兩顆一般寶石 → 1 次；兩顆「融合 1 次」再融合 → 2 次。另以 `leaves`（素材 5 階總數，雙方相加）記錄真實成本供拆解推算 |
| 屬性相容 | 融合後屬性種類**聯集 ≤ 2**（雙屬性只能配相同雙屬性、或含其屬性的單一寶石） |
| 同屬性數值 | `rnd(min(v1,v2), max(v1,v2) × 2)`（介於兩者間，上限較大值 2 倍） |
| 異屬性數值 | 各屬性 = `原值 × rnd(0.5, 1.5)`（不保證更高） |
| 失敗降解 | 較弱一方（單屬性優先；同為雙屬性比數值加總低者）消失，轉為同屬性低階寶石：50% 機率 4~8 顆 Lv1、50% 機率 2~4 顆 Lv2 |

## 9. 技能

### 9.1 基本數值

> 📈 2026-07-09 傷害重調：全部傷害技能的 base/per **×2.0 ~ ×3.0**（越強的技能加越多：
> 低耗速射 +100%、中型爆發 +120~150%、大招如隕石術/狂暴打擊 +180~200%）；
> 各技能實際數值見 `skills.js` 的 `SKILLS` 表。

| 項目 | 公式 |
|---|---|
| 技能原始傷害 | `(base + per × (等級-1))% × 對應攻擊力（物攻或魔攻）` |
| 一般技能法力消耗 | `round(原始消耗 × (1 + 0.1 × (技能等級-1)))`；非複利，例：50 → 55 → 60 |
| 多敵人技能傷害 | 當目標數 `N>1`：`原始傷害 × (1 + 範圍傷害%) ÷ N`；單一目標不套用範圍傷害加成，也不除以目標數 |
| 多段技能 | 上式為**每段**傷害，段數 `hits` 段各自結算（暴擊、閃避獨立判定） |
| 普攻目標 | 一次只攻擊一名敵人，不套用範圍傷害；目前鎖定第一名存活敵人，死亡後才切換 |
| 實際冷卻 | `技能冷卻 × (1 - 冷卻縮減%)` |
| 施放硬直 | `0.5 秒 × (1 - 施法速度%)`（延後下一次普攻） |
| 升級費用 | `⌊20000 × 當前等級 + 20^(1 + 當前等級/10)⌋` 金幣（上限 5,000,000；降級全額退還該級費用） |
| 等級上限 | 一般主動 `20 + 轉生次數×10`；被動 `30 + 轉生次數×10`；融合技 `素材等級加總 + 20 + 轉生次數×20` |
| 里程碑 | Lv.4 解鎖附加效果、Lv.8 強化該效果（覆蓋式，詳見 `skills.js` 的 `UNLOCKS` 表） |
| 技能樹門檻 | 第 N 階技能需先在該系投入 `N × 3` 點 |
| buff/治療縮放 | 皆為 `base + per × (等級-1)` 形式 |

特殊技能目前只保留主要增益效果；`timeWarp`、`blinkDodge`、`overload` 的第二個增益已移除。既有融合技能的相容清理由一次性存檔遷移處理，詳見根目錄 `ONE_TIME_MIGRATIONS.md`。

### 9.2 特殊效果公式

| 效果 | 公式 |
|---|---|
| 處決（execBelow） | 目標血量 < 門檻% 時，該段攻擊力 × execMult（預設 2） |
| 孤注一擲（gamble） | 傷害 × rnd(0.33, 1.67) |
| 雙重施法（doubleCastPct） | 機率追加 1 段 |
| 冰與火之歌（comboDetonate） | 目標同時處於減速＋燃燒 → 追加總傷害 × 比例%（無視防禦） |

| 治療 / 護盾技能 | 回復 `最大生命 × 比例%`；護盾見 §3.5 |

### 9.3 技能融合

| 項目 | 公式 / 規則 |
|---|---|
| 素材 | 2~4 個已學習的主動技能（被動與融合技不可） |
| 傷害合併 | 各素材 `base`、`per` × **0.75**（`FUSE_FACTOR`，2026-07-09 由 0.6 上調）後加總；屬性（物/魔）採多數決，任一真傷 → 真傷 |
| 元素合併 | 各元素占比 × 0.75 後加總 |
| DoT 合併 | 每種 × 0.7，同名取高，最多保留 2 種 |
| 控場合併 | 暈眩/減速時間取素材最大值 × 0.8 |
| 增益合併 | 從 `buff/buff2` 收集效果；同類型取素材當前實際數值較高者，去重後依首次出現順序最多保留 2 種；保留效果的 `base × 0.8`、`per` 與持續時間 |
| 減益合併 | 從 `debuff/debuff2` 收集效果；同類型取素材當前實際數值較高者，去重後依首次出現順序最多保留 2 種；保留效果的 `base × 0.8`、`per` 與持續時間 |

| 舊存檔融合技 | 一次性遷移加成（`skillDmgV2` 旗標）：舊 base < 400 → ×3、400~699 → ×3.5、≥ 700 → ×4（最高 +300%） |

| 初始等級 | 素材等級加總（素材歸零、點數轉移；刪除融合技全額歸還） |
| 變異觸發率 | `45% + 幸運值/3`（8 種變異，需與融合結果相關，至多 1 種） |

### 9.4 自動施放 AI

依裝載欄順序每 tick 至多施放一個：冷卻結束 → MP 足夠 → 符合條件（`hurt25~80` 血量門檻 / `debuffed` 有負面 / `shield` 護盾不足 5% / 傷害技需有目標 / 同名增益不重複疊放）。多敵人時，範圍技能取得全部存活敵人；普攻仍只鎖定第一名存活敵人。

## 10. 離線收益

```
有效離線時間 = min(實際離線秒數, 8 小時)（1 分鐘內不計）
期望暴擊倍率 = 1 + 暴擊率% × (暴擊傷害% - 1)
估算 DPS    = 物攻 × (1 - 當前階段怪物防禦減傷) × 攻速 × 期望暴擊倍率
單殺耗時    = 怪物血量 / DPS + 0.8 秒出怪間隔
擊殺數      = ⌊有效時間 / 單殺耗時 × 50% 效率⌋（上限 20,000）

金幣 = 怪物金幣 × 擊殺數 × (1 + 金幣加成%)
經驗 = 怪物經驗 × 擊殺數 × (1 + 經驗加成%)
裝備 = 依野外掉落表計算期望件數 ×(1+掉寶率)，高品質優先實體化最多 30 件、
       其餘折算碎片（每件 3 × 分解係數）
```

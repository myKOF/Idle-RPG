# 等級 / 轉生 系統解鎖總表（LV_upgrade_system）

> 本表只列「到了某個**等級 / 轉生次數 / 地圖進度**時，玩家會**解鎖什麼系統功能**，或觸發什麼**特規處理**」。
> 不含數值平衡、戰鬥公式、掉落率與掉落物等級表（那些請查 [`game_formula.md`](game_formula.md)）。

---

## 一、依「角色等級」解鎖

| 等級 | 解鎖 | 說明 |
|---|---|---|
| 每 **20 級** | 技能裝載欄 +1 格 | 可同時自動施放的技能數；初始 2 格，最多 20 格。 |
| **Lv.500** | 第 2 套裝備 | 達 500 級後開放第二套裝備的檢視、命名與切換。 |
| **Lv.2000** | 第 3 套裝備 | 達 2000 級後再開放第三套裝備的檢視、命名與切換。 |
| **Lv.9999** | 可轉生 | 達等級上限即可轉生（見「二、轉生」）。 |

> 神鑄系統另需搭配轉生，見下方「二」。

---

## 二、依「轉生次數」解鎖

轉生條件：等級達 **9,999** → 可轉生，最高 **10 轉**；轉生**保留**裝備、技能、資源、關卡進度，等級與經驗重置為 1。

| 轉生 | 解鎖 | 說明 |
|---|---|---|
| **1 轉** | 天賦系統 | 開放天賦頁；轉生後每升 1 級給 1 點轉生天賦點。 |
| **1 轉且 Lv.2000** | 神鑄系統 | 需同時達到 2000 級與 1 轉，開放後**永久保留**。 |
| 每 +1 轉 | 該轉 8 個天賦節點 | 1～10 轉逐轉解鎖，需「轉生次數 ≥ 該天賦轉數」。 |
| 每 +1 轉 | 熔爐 +1 座 | 0 轉 2 座，每轉 +1，上限 12 座。 |
| 每 +1 轉 | 技能等級上限提高 | 一般主動 +10／被動 +10／融合技 +20（每轉）。 |
| **3 轉** | 潛力技能分頁 | 技能頁出現「✨ 潛力」分類面板。 |
| 3 / 4 / 7 / 10 轉 | 潛力解鎖節點 | 對應轉數的「潛力啟示/覺醒/爆發/昇華」天賦；**目前整批置灰、暫停開放**。 |

## 三、依「地圖進度」解鎖

| 解鎖 | 條件 |
|---|---|
| 🏜️ 荒漠場景 | 草原打到第 **100 階** |
| 🦠 沼澤場景 | 荒漠打到第 **100 階** |
| 👑 菁英怪出現 | 每逢 **10 的倍數**階段 |
| 🗼 高塔三區：試煉之塔 1～50 / 地獄之塔 51～100 / 煉獄之塔 101～150 | 須**依序通關**前一層，才能挑戰下一層（另需付金幣） |

## 四、特規一次性處理（每位玩家一輩子各一次）

| 里程碑 | 等級窗口 | 處理機制 |
|---|---|---|
| 首件**獨特**裝備保底 | **30～33 級** | 窗口內每擊殺 1 隻野怪 +0.5% 累進機率，**窗口內保證必得 1 件「獨特」裝備**；若超過上限仍無任何獨特（含以上）裝備，下次擊殺直接補發。 |
| 首件**史詩**裝備保底 | **50～60 級** | 同上機制，保底 1 件「史詩」裝備。 |

> 發放為「指定稀有度、隨機部位」，等級依當時怪物等級。離線收益不計入，回線後首次擊殺會補發。程式：[`special_rules.js`](js/special_rules.js)。

## 五、技能 / 天賦 內部里程碑

| 門檻 | 效果 |
|---|---|
| 技能 **Lv.4** | 解鎖該技能的附加效果 |
| 技能 **Lv.8** | 強化該附加效果 |
| 技能樹第 N 階 | 需在該系先投入 `N × 3` 點才能學習 |
| 潛力解鎖天賦 **Lv.100 滿級** | 才固定解鎖對應的潛力技能名額（目前整批置灰） |

---

## 六、開局即有（無等級 / 進度門檻）

以下系統**一開始就能使用**：戰鬥掛機、技能與技能融合、裝備（強化/洗煉/分解/合成/附魔）、熔爐（2 座起）、寶石全套（鑲嵌/升階/合成/轉換/拆解/融合/商店）、高塔挑戰（受「金幣消耗」與「須依序通關」限制，不設等級門檻）。
